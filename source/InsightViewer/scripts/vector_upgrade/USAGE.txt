Vector upgrade toolkit for other TEST Neo4j databases

Prerequisites:
- export NEO4J_URI, NEO4J_USER, NEO4J_PASSWORD
- project venv exists at /home/robert/insightViewer/.venv
- for embeddings provider:
  - ollama: set OLLAMA BASE/MODEL in source/InsightViewer/config.ini (or config_private.ini)
  - openai: set OPENAI_API_KEY and choose --provider openai --model text-embedding-3-large

1) Create indexes/constraints only
cypher-shell -a "$NEO4J_URI" -u "$NEO4J_USER" -p "$NEO4J_PASSWORD" \
  -f /home/robert/insightViewer/source/InsightViewer/scripts/vector_upgrade/01_create_vector_indexes.cypher

2) Backfill embeddings only
/home/robert/insightViewer/.venv/bin/python \
  /home/robert/insightViewer/source/InsightViewer/scripts/vector_upgrade/02_backfill_chunk_embeddings.py \
  --provider ollama \
  --model mxbai-embed-large:latest \
  --batch-size 100

3) Run both steps with wrapper
/home/robert/insightViewer/source/InsightViewer/scripts/vector_upgrade/run_vector_upgrade.sh \
  --provider ollama --model mxbai-embed-large:latest --batch-size 100

4) One-command run using Docker Neo4j auth (no manual export)
/home/robert/insightViewer/source/InsightViewer/scripts/vector_upgrade/run_vector_upgrade_docker_auth.sh \
  --provider ollama --model mxbai-embed-large:latest --batch-size 100

Optional:
- choose non-default container name: export NEO4J_CONTAINER=my-neo4j
- override URI if needed: export NEO4J_URI=bolt://192.168.1.16:7687

Useful flags:
- --project <projectName>   process only one project
- --dry-run                 validate reads/embedding calls without writing
- --text-properties text,content,body,chunkText,value

Detailed repeatable checklist:
- /home/robert/insightViewer/source/InsightViewer/docs/vector_upgrade_checklist.txt

Post-upgrade validation:
1) Verify embedding coverage in Neo4j Browser
MATCH (c:Chunk)
RETURN count(c) AS total, count(c.embedding) AS withEmbedding, count(c)-count(c.embedding) AS missing;

2) Run end-to-end smoke checks (requires JWT token or ACCESS_TOKEN env)
cd /home/robert/insightViewer/source/InsightViewer
./scripts/smoke_retrieval_and_quiz.sh http://localhost:5001 <jwt_token>

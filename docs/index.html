Open-source knowledge graph viewer for Neo4j

See how knowledge, systems, decisions, and code connect.

InsightViewer is a lightweight web application for exploring connected knowledge. It started as a Neo4j graph viewer, but it is becoming a practical way to capture, visualize, and use organizational context.

View on GitHub Watch presentations Contact

Why InsightViewer?

Organizations do not only have data. They have applications, business rules, documents, meetings, decisions, dependencies, responsibilities, and technical implementation details. Much of this knowledge is scattered across tools and people.

InsightViewer helps turn that scattered knowledge into a graph that can be explored visually, extended with documentation, and used as context for people and AI systems.

The key point:
Models can reason. Experts can advise. But the organization must capture and connect its own knowledge, because that knowledge provides the memory for AI models.

What it can represent

Software architecture

Visualize dependencies between applications, pages, database objects, PL/SQL packages, procedures, and technical components.

Business logic

Connect business processes, rules, decisions, requirements, and implementation details in one navigable structure.

Meetings and decisions

Use structured meeting templates to capture topics, tasks, people, decisions, and follow-up actions as graph data.

Migration knowledge

Support legacy modernization by linking old systems, migration cases, target applications, documentation, and technical evidence.

Learning and research

Build personal or team knowledge graphs from notes, concepts, sources, examples, and explanations.

AI-ready context

Attach chunks, summaries, and documentation to graph nodes so AI can answer questions with better organizational context.

Core idea

InsightViewer is not only a graph visualization tool. It is a way to make knowledge visible, connected, and reusable.

Explore the graph

Navigate connected data interactively, expand nodes, follow dependencies, and move from overview to detail.

Add meaning to nodes

Attach HTML content to graph nodes: explanations, images, links, templates, summaries, or business documentation.

Use templates

Generate repeatable views and queries so users do not need to write Cypher for every common question.

Prepare for AI

Combine graph structure, document chunks, full-text search, vector search, and generated Cypher as complementary ways to find answers.

Built for practical use

InsightViewer is designed for situations where Neo4j Browser is too technical for many users and enterprise graph visualization tools may be out of reach. It uses familiar web technologies and can be adapted to different graph models.

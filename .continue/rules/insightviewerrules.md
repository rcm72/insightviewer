# Project overview

- This is a Python project that uses Neo4j.
- Neo4j is accessed using the official Python driver (`neo4j`).
- Backend code is in `source/InsightViewer/app/models` or `source/InsightViewer/app/app.py`
- Frontend code is in `source/InsightViewer/app/templates` or `source/InsightViewer/app/static`
- Graph / database-related code is likely in `backend/db/` or `source/InsightViewer/app` .
- Ignore `.venv/`, `node_modules/`, `dist/`, and `build/`.
